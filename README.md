# What is ZombieOS?

ZombieOS (ZOS) is a futuristic operating system ecosystem currently in active development.

The current public builds are prototype runtime environments built on top of existing operating systems while the long-term goal is a fully independent operating system experience.

ZombieOS is designed around:
- a connected ecosystem
- integrated social features
- creator and developer programs
- futuristic UI/UX
- cross-platform accessibility
- custom services and APIs
- experimental runtime systems
- future kernel-level operating system development

The ZombieOS ecosystem includes:
- the main website
- account systems
- dashboard systems
- support systems
- social platform infrastructure
- future desktop applications
- future mobile applications
- future native operating system releases
