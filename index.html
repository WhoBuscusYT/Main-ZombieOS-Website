What is ZombieOS?

ZombieOS (ZOS) is a futuristic operating system and runtime platform built around customization, performance, and the Z# programming language.

The current prototype runs on existing operating systems while development continues toward a standalone kernel version capable of powering the full ZombieOS experience.

ZombieOS includes the Z# language, developer tools, account system, support platform, community features, and future ecosystem services.
